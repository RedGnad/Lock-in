# Lock In V5 product model

V5 is competitive accountability, not a general betting market. Friends choose one self-improvement mission, equal stake, daily target, duration and required number of successful days. Finishers split the pool; Lock In takes no fee.

## Pools

Each pact is its own pool. A 3-day Strava pact never competes with a 7-day or Duolingo pact. Configuration is immutable after creation. The 3-day template is onboarding and still requires all three days, so a user cannot join immediately before one already-planned activity and finish the whole challenge.

## Missions

- Strava: 1, 3, 5 or 10 km GPS Run; 3/3, 5/7, 10/14 or 20/30 successful days.
- Duolingo: 10, 20, 30 or 50 new XP; the same duration templates.

XP is never a leaderboard score because exercise types and bonuses differ. It is a binary daily threshold. Run distance is likewise a threshold, not winner ranking.

## Payouts

Every wallet stakes the same amount. Finishers divide the exact escrow pool. If no one finishes, everyone is refunded. This prevents the protocol from taking abandoned stakes and keeps the competition capital-proportional rather than wallet-count weighted.

## Admission

The public app remains limited to a small 18+ beta cohort, max 1 USDC per participant/pact, and paused by default. Opening requires both provider canaries and contract/security review. Strava and Duolingo are independent external services, not sponsors or partners.
